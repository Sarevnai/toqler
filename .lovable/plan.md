

# Trocar imagem do Hero

Substituir a imagem atual do hero da landing page (`src/assets/dashboard-preview.png`) pela screenshot do dashboard enviada.

## Passos

1. Copiar o arquivo enviado (`user-uploads://screencapture-...png`) para `src/assets/dashboard-preview.png`, sobrescrevendo o atual
2. Nenhuma mudanca de codigo necessaria -- o `HeroSection.tsx` ja importa esse arquivo

## Resultado

A landing page exibira a screenshot atualizada do dashboard com o design system Pluma aplicado.
